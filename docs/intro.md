# 一人企业的创业者模式

这是一个让我更舒适的创业者模式，我称之为“一人企业”。

它的核心是，你可以在不招募任何人的情况下，通过自己的努力，建立一个可以持续运营的企业。

虽然这个模式存在着一些局限性和不确定性，我将会在接下来的 1-2 年时间内，尝试用这个模式来创业。

## 为什么要创业

我认为，创业是一种非常有趣的生活方式，它可以让你在不断的挑战中，不断的成长。

## 为什么要一人创业

拒绝盲目增长，坚持小规模发展。

## 如何保证技术的可行性

你不必成为一个领域的世界级专家才算擅长。人们通常最喜欢从只比他们稍微领先几步的人那里学习。
